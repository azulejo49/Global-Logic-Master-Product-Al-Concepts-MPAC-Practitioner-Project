---

## Project Overview

This project is a **capstone AI trading dashboard** built for paper trading and educational purposes.  
It demonstrates:

- Realistic candlestick charts with **SMA, EMA, RSI, and volume indicators**.
- **Volatility regime detection** (low, medium, high) using rolling standard deviation.
- **LLM-style AI narrative** for market insights and trend bias.
- Adaptive indicator windows based on dataset length (demo, reduced history, full production).

> ⚠️ Paper trading only. **No financial advice is provided.**

## CI Build Script (`ci_build.py`)

The `ci_build.py` script automates:

1. **Folder and file creation** for the full project structure.  
2. **Virtual environment setup** (`venv`).  
3. **Installation of dependencies** (`streamlit`, `pandas`, `plotly`, `ta`).  
4. **Zipping the project** into `MPAC_AI_Trading_Capstone.zip` for distribution or reproducibility.  

### Usage

```bash
# Run the CI build
python ci_build.py

# Navigate to project folder
cd MPAC_AI_Trading_Capstone

# Activate virtual environment
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# Run the app
cd app
streamlit run app.py
